# Danielmschmidt Tap

## How do I install these formulae?

`brew install danielmschmidt/tap/<formula>`

Or `brew tap danielmschmidt/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "danielmschmidt/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
